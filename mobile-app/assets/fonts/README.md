# San Francisco Fonts

This directory should contain the San Francisco font files:

## Required Files:

- SF-Pro-Display-Regular.ttf
- SF-Pro-Display-Medium.ttf
- SF-Pro-Display-Semibold.ttf
- SF-Pro-Display-Bold.ttf
- SF-Pro-Text-Regular.ttf
- SF-Pro-Text-Medium.ttf
- SF-Pro-Text-Semibold.ttf
- SF-Pro-Text-Bold.ttf

## Where to get them:

1. Download from Apple Developer portal (requires Apple Developer account)
2. Extract from macOS system fonts (if you have a Mac)
3. Use alternative system fonts like -apple-system or San Francisco alternatives

## Alternative Configuration:

If San Francisco fonts are not available, the app will fall back to:

- iOS: Native San Francisco fonts (built-in)
- Android: Roboto (system default)
- Web: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto

## Usage:

- SF Pro Display: For larger text, headings, titles
- SF Pro Text: For body text, smaller text elements
